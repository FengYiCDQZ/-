\part{分析与总结}
向量的向量积有着广泛的用处
\begin{itemize}
  \item 面积
  \item 体积
  \item 法向量
\end{itemize}
下面将以3几道例题来说明

\prob 已知椭圆$\frac{x^2}{a^2}+\frac{y^2}{b^2}=1(a>b>0)$上有两动点$A$、$B$，求$S_{\triangle OAB}$最大值\\
\from{方若愚}$\textrm{设}A(a\cos\alpha,b\sin\alpha),B(a\cos\beta,b\sin\beta)\\
\so S_{\triangle OAB}=\frac{1}{2}\abs{\vv{OA}\times\vv{OB}}=\frac{1}{2}\abs{ab\sin\beta\cos\alpha-ab\sin\alpha\cos\beta}\\
=\frac{1}{2}ab\abs{\sin(\alpha-\beta)}\leq\frac{1}{2}ab$
